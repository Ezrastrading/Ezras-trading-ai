# Supabase production migrations — apply in order

## CRITICAL (required for live trade_events sync + edge metadata used by the app)

**Convenience:** A single combined file (same order: 1 then 2 then 3) is in the repo as `ALL_REQUIRED_LIVE_MIGRATIONS.sql` — paste its full contents into SQL Editor, or run each file separately below.

1. trade_intelligence_databank.sql  
   Creates `trade_events` (full row shape used by Trade Intelligence Databank), daily/weekly rollups, strategy/avenue summaries.  
   **If missing:** all `trade_events` upserts fail; deployment proof and reality-layer trade truth break.

2. edge_validation_engine.sql  
   Adds `edge_id`, `edge_lane`, `market_snapshot_json` on `trade_events`; creates `edge_registry`.  
   **If missing:** upserts that include edge columns fail; edge registry mirror unavailable.

3. trade_events_acco_columns.sql  
   Adds ACCO / spot / options / latency columns on `trade_events`.  
   **Requires:** step 2 applied first (so `edge_id` exists).  
   **If missing:** rows with `instrument_kind`, `base_qty`, etc. may fail on insert.

## OPTIONAL (mirrors / learning — safe to skip for minimal trade sync)

4. balance_snapshots_milestones.sql — treasury / milestone tables.  
5. lessons_progression.sql — lessons, progression, ceo_briefings.
6. rls_security_advisor_fix.sql — RLS + least-privilege policies (backend/service-role only) for public mirror tables; removes always-true policies.

## Verification

After applying 1–3, the deployment runner’s remote probe must succeed: a `SELECT` of all columns the app sends to `trade_events` must not error.

## Troubleshooting (HTTP 404 on `/rest/v1/trade_events`)

- **Wrong project:** `SUPABASE_URL` and the API key must refer to the **same** Supabase project (Dashboard → Settings → API).
- **Missing table:** Apply steps **1–3** above in this database; 404 often means `trade_events` was never created here.
- **Repo vs DB:** Migrations live under `supabase/*.sql` in the repo — they do not apply themselves; operators must run the SQL against the target project.
